# docs

Documentación técnica de referencia del proyecto. Acá van archivos que proveen contexto a los agentes de IA sobre APIs, requisitos académicos y recursos externos.

## Archivos activos (usados para desarrollar el proyecto)

- `BACKEND_API_REFERENCE.md` — Referencia de la API del backend con el que se comunica la app.
- `BACKEND_GITHUB_REPO.txt` — Link y contexto del repositorio GitHub del backend, para investigación más profunda.
- `RN_CLASSES_CONTEXT.md` — Requisitos y contexto de la cátedra (clases de React Native exigidas por la materia).

## Archivos de referencia (contexto histórico, no se usan directamente)

- `UNITREE_GO2_API_REFERENCE.md` — API HTTP/WebSocket del robot cuadrúpedo Unitree Go2. Queda como referencia de la API de bajo nivel del robot.
- `UNITREE_G1_API_REFERENCE.md` — API HTTP/WebSocket del robot humanoide Unitree G1. Queda como referencia de la API de bajo nivel del robot.
